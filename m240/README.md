# M240 Flamethrower Build Guide

Welcome to the M240 Flamethrower build documentation. This guide contains comprehensive assembly instructions for building the M240 Incinerator Unit from Aliens.

## Table of Contents

### Assembly Guide
- [Assembly M240](assembly-m240.md) - Complete M240 assembly instructions with exploded views

## Build Overview

The M240 Flamethrower is a prop replica of the iconic incinerator unit used in the film Aliens. This build features custom 3D printed components with copper pipe integration for an authentic look.

## Parts Required

### Hardware
- **M3 x 15mm screws** - 8 pieces (round head)
- **M3 x 8mm screws** - 5 pieces (round head)
- **M3 x 8mm screws** - 2 pieces (countersunk)
- **M3 x 4mm screws** - 3 pieces
- **M3 x 20mm screws** - 1 piece
- **Small pen spring** - 1 piece (for trigger)

### Copper Pipe (12.7mm / 1/2 inch diameter)
- **390mm length** - 1 piece
- **321mm length** - 1 piece

## Getting Started

1. Review the [Assembly M240](assembly-m240.md) guide for complete build instructions
2. Gather all required hardware and copper pipe lengths
3. Follow the exploded view diagrams for proper component placement
4. Install the trigger spring mechanism
5. Secure all assemblies with appropriate screws

## Key Features

- Front assembly with copper pipe integration
- Rear housing with Hansen coupler
- Left and right trigger housing assemblies
- Detailed exploded views for assembly guidance

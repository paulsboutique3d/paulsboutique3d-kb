# M41A MK2 Pulse Rifle Build Guide

Welcome to the M41A MK2 Pulse Rifle build documentation. This guide contains assembly instructions, spring specifications, and component details for building the MK2 variant of the M41A Pulse Rifle replica.

## Table of Contents

### Assembly & Components
- [M41A MK2 Thompson](m41a-mk2-thompson.md) - Thompson receiver assembly for MK2 variant
- [M41A MK2 Breech Bolt](m41a-mk2-breech-bolt.md) - Breech bolt assembly and spring specifications
- [M41A MK2 Shrouds](m41a-mk2-shrouds.md) - Shroud assembly for MK2 variant
- [M41A MK2 U1 Springs](m41a-mk2-u1-springs.md) - Spring specifications and assembly for U1 components

## Build Overview

The M41A MK2 is an updated variant of the M41A Pulse Rifle with refined internal mechanisms and improved spring configurations for better functionality and reliability.

## Key Specifications

### Spring Sizes
- **Breech Bolt Spring**: 4.5mm OD x 0.5mm thickness
- **U1 Hammer Spring**: 9-9.5mm OD x 0.5mm thickness
- **Trigger Spring**: 6mm OD x 0.5mm thickness
- **Gate Spring**: 4.5mm OD x 0.5mm thickness

## Getting Started

1. Start with the [M41A MK2 Thompson](m41a-mk2-thompson.md) guide for the main receiver assembly
2. Follow the [M41A MK2 Breech Bolt](m41a-mk2-breech-bolt.md) guide for breech assembly
3. Install shrouds following the [M41A MK2 Shrouds](m41a-mk2-shrouds.md) guide
4. Reference [M41A MK2 U1 Springs](m41a-mk2-u1-springs.md) for all spring installation details

# M41A/2 Pulse Rifle Build Guide

Welcome to the M41A/2 Pulse Rifle build documentation. This guide contains assembly instructions for the M41A/2 variant, featuring refined internal mechanisms and updated component designs.

## Table of Contents

### Assembly Guides
- [M41A/2 Thompson Inner](m41a-2-thompson-inner.md) - Thompson receiver and internal mechanism assembly
- [M41A/2 Remington](m41a-2-remington.md) - Remington shotgun component assembly
- [Assembly – Shrouds](assembly-shrouds.md) - Shroud and front vent assembly

## Build Overview

The M41A/2 is a variant of the M41A Pulse Rifle with improved internal mechanisms and refined component fit. This version features updated Thompson internals and enhanced Remington integration.

## Getting Started

1. Begin with the [M41A/2 Thompson Inner](m41a-2-thompson-inner.md) guide for the main receiver and internal components
2. Follow the [M41A/2 Remington](m41a-2-remington.md) guide for the shotgun component assembly
3. Install shrouds and front vent following the [Assembly – Shrouds](assembly-shrouds.md) guide

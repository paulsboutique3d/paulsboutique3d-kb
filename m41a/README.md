## Build Overview

The M41A Pulse Rifle is a combination of Thompson submachine gun and Remington 870 shotgun components with custom shrouds and electronics. This build includes functional trigger mechanisms, LED lighting, sound effects, and a digital round counter.

## Getting Started

1. Review the [BOM Listing](bom-listing.md) to ensure you have all required parts
2. Begin with the [Assembly – Thompson](assembly-thompson.md) guide for the main receiver
3. Follow the [Assembly – Remington](assembly-remington.md) guide for the shotgun component
4. Install shrouds following the [Assembly – Shrouds](assembly-shrouds.md) guide
5. Add electronics per the [M41A Electronics Install](m41a-electronics-install.md) guide
6. Finish with paint using the [Paint Colours](paint-colours.md) specifications

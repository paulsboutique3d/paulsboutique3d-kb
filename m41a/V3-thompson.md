
The V3 Thompson is different from V2 in these areas:

1. The upper reciever is lower, matching original Thompson specifcations - 36.367mm
2. The Thomspn barrel line is now at: 23.5mm which matches original Thompson and Airosft spec.
3. The lower frame has an additional tab that makes mating the handle easier and more sturdy.
4. The Thompson bolt is lower - dimension height 32.78mm
5. The Thompson mag is now dimensionally the same as Tokyo Marui Metal Airsoft magazines.

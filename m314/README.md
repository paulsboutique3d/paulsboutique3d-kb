## Build Overview

The M314 Motion Tracker is built around a modified Kango hammer drill body with various photographic equipment donor parts including a Vivitar 2000 flash unit, Hama camera accessories, and custom 3D printed components. This replica combines vintage equipment with modern fabrication techniques to create an accurate representation of the Motion Tracker.

## Donor Parts Required

### Main Components
- Kango hammer drill body
- Vivitar 2000 flash unit
- Hama camera accessories (remote release, mounting bracket)
- T28 component
- Jobo mini light
- Self timer mechanism
- Pump mechanism
- Matchbox component

## Getting Started

1. Start with the [M314 Kango Assembly](m314-kango-assembly.md) guide for the main body and trigger mechanism
2. Follow the [Vivitar, Hama and T28](vivitar-hama-and-t28.md) guide for the main display components
3. Install the [Hama Remote Release](hama-remote-release.md) cable assembly
4. Add the [Jobo Mini Light](jobo-mini-light.md) attachment
5. Assemble the [Pump and Matchbox](pump-and-matchbox.md) components
6. Complete with the [Self Timer and Case](self-timer-and-case.md) assembly

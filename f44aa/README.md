## Build Overview

The F44AA is a detailed prop replica featuring working trigger mechanisms, LED lighting, dual IPS screens, sound effects via Dayton speaker, and multiple spring-loaded components. This build combines precision machining with 3D printed parts for an authentic representation.

## Key Components

### Electronics
- 4x Microswitches
- 2x 1.9 inch IPS screens
- 1x WS2812 RGB LED
- 1x Dayton speaker

### Springs Required
- Wing springs (2x): 10mm x 6mm x 0.6mm
- Stock lock (1x): 32mm x 5mm x 0.6mm
- Trigger spring (1x): 26mm x 7mm x 0.6mm
- Safety spring (1x): 22mm x 6mm x 0.6mm
- Magazine spring (1x): 6mm x 9mm x 0.6mm
- Breech spring (1x): 130mm x 7mm x 0.6mm
- Screen springs (2x): Various sizes

### Materials
- 3.16mm brass rod (500mm total)
- 5.56mm brass tube (158mm)
- M5 threaded rod (172mm)

## Getting Started

1. Review the [F44AA BOM](f44aa-bom.md) to ensure you have all required parts
2. Begin with the [F44AA Base Assembly](f44aa-base-assembly.md) guide for the main body
3. Follow the [F44AA Trigger Assembly](f44aa-trigger-assembly.md) for trigger mechanism
4. Install the [F44AA Rear Piston Assembly](f44aa-rear-piston-assembly.md)
5. Complete the [F44AA Rear Stock](f44aa-rear-stock.md) with wing springs
6. Add shrouds per the [F44AA Shrouds](f44aa-shrouds.md) guide
7. Install electronics following [F44AA Light](f44aa-light.md) guide
8. Complete with [F44AA Target Screen](f44aa-target-screen.md) installation
9. Finish with [F44AA Mag Assembly](f44aa-mag-assembly.md)
